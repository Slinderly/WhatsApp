API Reference

Endpoints disponibles para integrarlo en tu proyecto

Sesiones

GET/wa/sessions/:accountId

Lista todas las sesiones de una cuenta con su estado y dispositivo.

{
  "sessions": [
    {
      "sessionId": "abc123",
      "status": "connected",
      "device": {
        "phone": "5219991234567",
        "name": "Mi iPhone",
        "connectedAt": "2025-01-01T00:00:00.000Z"
      }
    }
  ]
}
GET/wa/status/:accountId/:sessionId

Estado de una sesión. Si hay QR disponible lo devuelve como data URL.

{
  "sessionId": "abc123",
  "status": "qr_ready",
  "device": null,
  "hasQR": true,
  "qr": "data:image/png;base64,..."
}

Conectar

POST/wa/connect/qr

Inicia sesión con flujo QR. Luego haz polling a /wa/status para obtener el código.

// Body
{ "accountId": "cuenta1", "sessionId": "opcional" }
// Response
{ "success": true, "sessionId": "abc123" }
POST/wa/connect/pairing

Inicia sesión con código de emparejamiento. Devuelve el código de 8 dígitos.

// Body
{ "accountId": "cuenta1", "phoneNumber": "+5219991234567" }
// Response
{ "success": true, "sessionId": "abc123", "code": "ABCD-1234" }
DELETE/wa/sessions/:accountId/:sessionId

Desconecta y elimina la sesión. Borra los archivos de autenticación.

Mensajes

POST/wa/send

Envía un mensaje de texto. Usa la primera sesión conectada del accountId.

// Body
{
  "accountId": "cuenta1",
  "jid": "5219991234567@s.whatsapp.net",
  "text": "Hola!",
  "sessionId": "opcional"
}
// Response
{ "success": true }
GET/wa/messages?accountId=cuenta1&limit=50

Log de mensajes recibidos en memoria (demo). En producción escucha el evento message del SDK y guarda en tu DB.

Integración Node.js

const wa = require('./src/whatsapp');

// Escuchar mensajes entrantes
wa.on('message', (msg) => {
  // msg = { accountId, sessionId, jid, from, text, timestamp, raw }
  console.log(msg.from, ':', msg.text);
  // Aquí conectas tu IA, DB, lógica de negocio, etc.
});

wa.on('connected',    ({ accountId, sessionId, phone }) => { ... });
wa.on('disconnected', ({ accountId, sessionId, reason }) => { ... });

// API programática
await wa.connectQR('mi-cuenta', 'sesion-1');
await wa.connectPairing('mi-cuenta', 'sesion-2', '+521234567890');
await wa.sendMessage('mi-cuenta', '521234567890@s.whatsapp.net', 'Hola');
wa.disconnectSession('mi-cuenta', 'sesion-1');

// Al arrancar el servidor
wa.restoreSessions(); // reconecta sesiones guardadas
