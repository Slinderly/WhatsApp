API Reference

Endpoints disponibles para integrarlo en tu proyecto

Sesiones

GET/wa/sessions/:accountId

Lista todas las sesiones de una cuenta con su estado y dispositivo.

GET/wa/status/:accountId/:sessionId

Estado de una sesión. Si hay QR disponible lo devuelve como data URL.

Conectar

POST/wa/connect/qr

Inicia sesión con flujo QR.

{ "accountId": "cuenta1", "sessionId": "opcional" }
POST/wa/connect/pairing

Inicia sesión con código de emparejamiento.

{ "accountId": "cuenta1", "phoneNumber": "+5219991234567" }
DELETE/wa/sessions/:accountId/:sessionId

Desconecta y elimina la sesión.

Mensajes

POST/wa/send

Envía un mensaje de texto.

{ "accountId": "cuenta1", "jid": "521234@s.whatsapp.net", "text": "Hola!" }
GET/wa/messages?accountId=cuenta1&limit=50

Log de mensajes recibidos en memoria.

Grupos

GET/wa/groups/:accountId

Lista todos los grupos en los que está la sesión.

{ "groups": [{ "id": "123@g.us", "name": "Grupo", "participants": 42 }] }
GET/wa/groups/:accountId/:groupId/members

Lista todos los miembros de un grupo (excluye tu propio número).

Difusión

POST/wa/broadcast

Inicia una difusión privada a los miembros de un grupo.

{ "accountId": "cuenta1", "groupId": "123@g.us", "message": "Hola!", "delayMs": 6000 }
GET/wa/broadcast/status

Estado actual de la difusión en curso.

POST/wa/broadcast/stop

Detiene la difusión en curso.

IA

GET/ai/settings

Obtiene la configuración actual de la IA.

POST/ai/settings

Actualiza configuración: apiKey, enabled, systemPrompt, model, maxHistory.
