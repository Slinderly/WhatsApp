API Reference

Endpoints REST disponibles

Sesiones

GET/wa/sessions/:accountId

Lista sesiones con estado y dispositivo.

GET/wa/status/:accountId/:sessionId

Estado + QR como data URL si está disponible.

DELETE/wa/sessions/:accountId/:sessionId

Desconecta y elimina la sesión.

Conectar

POST/wa/connect/qr
{ "accountId": "cuenta1" }
POST/wa/connect/pairing
{ "accountId": "cuenta1", "phoneNumber": "+521234567890" }

Mensajes

POST/wa/send
{ "accountId": "cuenta1", "jid": "521234@s.whatsapp.net", "text": "Hola!" }
GET/wa/messages?accountId=cuenta1

Log de mensajes en memoria.

Grupos & Difusión

GET/wa/groups/:accountId

Lista grupos de la cuenta.

GET/wa/groups/:accountId/:groupId/members

Miembros del grupo (excluye tu número).

POST/wa/broadcast
{ "accountId": "cuenta1", "groupId": "123@g.us", "message": "Hola!", "delayMs": 6000 }
GET/wa/broadcast/status

Estado de la difusión en curso.

POST/wa/broadcast/stop

Detiene la difusión.
